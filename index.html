MorseDecoder

Developed the backend of a Morse code decoder in Java, implementing a binary tree structure and a recursive algorithm. This backend efficiently translates Morse code signals into human-readable text. Users can interact with the decoder by providing Morse code sequences through user input, and the program utilizes the recursive algorithm to decode the signals character by character.
